# Prontuvia v1.4.3 — tipografia sem alteração de layout

Esta versão reverte integralmente o redesign estrutural da v1.4.2 e preserva o
layout original do Prontuvia.

## Alterações exclusivas na escrita

- Fonte Nunito Sans incorporada localmente.
- Título de saudação reduzido para 30 px e peso 700.
- Pesos e tamanhos equilibrados em menus, métricas, tabelas e formulários.
- Melhor altura de linha e espaçamento entre letras.
- Textos secundários mais legíveis sem aumentar cartões ou mover elementos.

Não há alterações de banco de dados nem nova migração.
