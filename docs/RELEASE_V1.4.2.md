# Prontuvia v1.4.2 — redesign do painel

Esta atualização substitui a composição visual anterior por um painel claramente
novo, sem alterar o banco de dados.

## Mudanças visíveis

- Menu lateral claro, com item ativo em gradiente verde.
- Painel de boas-vindas amplo, com data e fundo orgânico.
- Métricas com ícones próprios e cores por finalidade.
- Novo cartão de jornada assistencial.
- Maior contraste entre títulos, textos e dados secundários.
- Espaçamento, sombras, bordas e proporções reconstruídos.
- Indicador `Versão 1.4.2` no rodapé do menu.
- Fontes DM Sans e Manrope permanecem incorporadas localmente.

Não há nova migração. Após substituir os arquivos, execute `npm install`, reinicie
API e interface e force a atualização do navegador.
