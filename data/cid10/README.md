# Base CID-10 brasileira

O arquivo `CID-10-SUBCATEGORIAS.csv` contém as subcategorias da CID-10
disponibilizadas pelo DATASUS (versão 2008), convertidas de Windows-1252 para
UTF-8 para uso local no Prontuvia.

Fonte oficial: https://www2.datasus.gov.br/cid10/V2008/descrcsv.htm

O Prontuvia mantém a versão da classificação no resultado da pesquisa. A
CID-11 não é copiada para este diretório: ela é consultada na API oficial da
OMS com credenciais configuradas no servidor.
